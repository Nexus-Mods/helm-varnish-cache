# Varnish Helm Charts

This repository hosts a Helm chart for Varnish Cache.

## Usage

``` shell
$ helm install -f values.yaml release-name oci://registry-1.docker.io/varnish/varnish-cache
```

See also [Configurations](docs/configurations.md).
